# Vagrant Box
A vagrant box for web development.

## Status
Work in progress.

## Contents
* Apache 2.4
* PHP 5.5 (FPM)
* MySQL 5.6
* Cron
* Dovecot + Postfix
* Image Magick
* Locale
* SQLite
* Zip

## Prerequisites
* Vagrant >= 1.6.0
* VirtualBox or VMWare Workstation/Fusion
* Librarian Chef

## Installation
* Clone the repository
* Run `librarian-chef install` to install cookbooks
* Run `vagrant up` to run the initial provisioning
