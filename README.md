# Vagrant Box
A vagrant box for web development.

## Status
Work in progress.

## Contents
* Apache 2.2
* Cron
* Dovecot + Postfix
* Git
* Image Magick
* Locale
* MySQL
* PHP 5.4 FPM
* SQLite
* Zip

## Prerequisites
* Vagrant
* Ubuntu 13.04 64bit base-box named "raring64"
* VirtualBox or VMWare Workstation/Fusion
* Librarian Chef

## Installation
* Clone the repository
* Run `librarian-chef install` to install cookbooks
